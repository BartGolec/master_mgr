Application manual for certain Actions
See all data
Allows to see the data for every single node stored in database.
See data for certain node
Allows to see the data for specific node stored in database
Add data manually
Allows to add single data object for certain node stored in database. It is helpful when node lacks some of the power grid data.
Upload a file
Allows to upload a CSV format file with power grids parameters data to database. After that, application creates plots for every parameter chosen by user. Also it calculates minimal, maximal and average value for every parameter. Time period of certain data measurements is presented. Below all the plotted charts user has ability to overview average powers for all the data from uploaded file, calculated by the application.
OpenDSS interaction
Main component of the application. First window specifies the OpenDSS model to interfere with. It requires its directory path from local machine. After that all the data from given model is presented with options to add new loads to the model or to edit current one. Whenever loads are properly prepared, Voltage waveforms can be plotted by clicking "Plot voltage for certain model". The form requires directory for both OpenDSS model and OpenDSS executable file from local machine. When provided, application communicates with OpenDSS and voltage chart for certain nodes is visualised.
